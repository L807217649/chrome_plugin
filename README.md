### 提交数据的方式

1 本数据根据人工进行了a-z的区分，所以需要你来区分一下对应name位置。

  比如：慕课网--M; W3c--W; 数字或者特殊字符--jing。

```
            {
                "name": "365psd",  //必填
                "url": "http://365psd.com/",  //必填
                "des": "兢兢业业每天更新着用户界面相关的PSD",  //选填
                "tags": [
                    "界面设计"   //必填，至少一个
                ]
            }

```

2， 新建一个issue，在issue中填上数据即可。
